# Twocan Africa – Technical Management System

A web-based radio and SIM card tracking system for the Twocan Africa Technical Department.

## Features
- Radio repair tracking with status updates
- Sales register
- SIM card inventory management
- Real-time data sync across all devices
- Works offline

## How to Use
1. Open the app link in any browser
2. Log in with your admin credentials
3. Use the tabs to navigate between Repairs, Sales, and SIM Inventory

## Access
This system is for authorized Twocan Africa staff only.
